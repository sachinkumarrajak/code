Palindrome number both in c and java

If a number remains same, even if we reverse its digits then the number is known as palindrome number. For example 12321 is a palindrome number because it remains same if we reverse its digits.

  • Java Code

Program for palindrom no in C

#include <stdio.h>
#include <conio.h>
void  main()
{
    int n, m, a = 0,x;
   clrscr();
   printf("Enter any number:");
   scanf("%d",&n);
 
    m = n;
 
    while(n > 0)
        {
            x = n % 10;
            a = a * 10 + x;
            n = n / 10;
        }
        if(a == m)
	{	printf("yes number is Palindrome");
        }
        else
        {
	    printf(" number is not palindrome 	" );
	}
		getch();
}
